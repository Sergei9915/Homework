Список команд

  1. git --version - проверить версию установленного гита;
  2. git config --global user.name - указывает имя автора;
  3. git config --global user.email - указывает электронную почту автора;
  4. git config --list - показывает список настроек гита;
  5. git status - показывает изменения в репозиторий;
  6. git add - добовляет файл в репозиторий;
  7. git commit -m - закомитеть файл;
  8. git push - загружает все изменения в удаленный репозиторий;
  9. git pull - загружает историю из удаленного репозитория;
  10. git clone - скачивает репозиторий вместе со всей его историей изменений.
